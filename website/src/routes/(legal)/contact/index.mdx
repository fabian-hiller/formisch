# Contact

Formisch is an open source project led by Fabian Hiller and developed publicly on GitHub by other contributors. The project is non-commercial and the source code is available under the MIT licence.

## General inquiries

For general inquiries, such as bug reports and feature requests, please <a href={`${import.meta.env.PUBLIC_GITHUB_URL}/issues/new`} target="\_blank" rel="noreferrer">create an issue</a> on GitHub. If you have any questions or need help with Formisch, you can <a href={import.meta.env.PUBLIC_DISCORD_URL} target="_blank" rel="noreferrer">ask the community</a> on our Discord server.

## Security issues

To report a vulnerability, please contact us via hello@fabianhiller.com. We recommend that you use the latest versions of the library to ensure that your application remains as secure as possible.

## Partner requests

If you are interested in a partnership with Formisch, please contact us at hello@fabianhiller.com. We are open to collaborations that help us improve the project and make it more accessible to a wider audience.
