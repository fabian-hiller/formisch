# getAllErrors

Retrieves all error messages from all fields in the form by walking through the entire field store tree. This is useful for displaying a summary of all validation errors across the form.

```ts
const errors = getAllErrors(form);
```

## Parameters

- `form` <Property {...properties.form} />

### Explanation

The `form` parameter is the form store to retrieve errors from. The function walks through the entire field store tree to collect all validation error messages from every field.

## Returns

- `result` <Property {...properties.result} />
